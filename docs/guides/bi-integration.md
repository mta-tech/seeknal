The BI Integration guide will teach you to:

1. **Connect BI Tools** — Tableau, Power BI, Metabase, Looker, Superset
2. **Configure Data Sources** — ODBC/JDBC connections
3. **Build Dashboards** — Create visualizations with semantic models
4. **Enable Self-Serve Analytics** — Empower business users

- Completed semantic models and metrics deployment
- Basic understanding of BI tools
- Access to StarRocks or other warehouse
